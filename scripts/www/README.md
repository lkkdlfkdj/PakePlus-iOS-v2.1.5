# 隐患上报 App

根据 apec-kit 规则开发的隐患上报应用，支持位置定位、照片上传、权属类别选择和隐患类型选择等功能。

## 功能特点

1. **隐患位置**：
   - 支持获取当前地理位置（需要浏览器授权）
   - 可上传位置相关照片

2. **权属类别**：
   - 市政
   - 电力
   - 通信
   - 水务
   - 燃气
   - 其他

3. **隐患类型**：
   - 丢失
   - 破损
   - 断裂
   - 松动
   - 沉降
   - 高低不平
   - 支持详细说明

4. **现场照片**：
   - 支持多照片上传
   - 实时照片预览
   - 适配移动设备拍照功能

## 技术实现

- **前端技术**：HTML5 + CSS3 + JavaScript
- **响应式设计**：适配桌面端和移动端
- **地理位置**：使用浏览器 Geolocation API
- **照片上传**：使用 FileReader API 实现本地预览
- **地图服务**：集成高德地图 API 进行逆地理编码（需要替换为实际 API 密钥）

## 使用方法

1. **本地运行**：
   - 直接打开 `index.html` 文件即可在浏览器中运行
   - 建议使用现代浏览器（Chrome、Firefox、Safari、Edge 等）

2. **部署到服务器**：
   - 将所有文件上传到您的服务器
   - 通过服务器地址访问应用

3. **配置说明**：
   - 在 `script.js` 文件中，需要替换 `apiKey` 为您的高德地图 API 密钥
   - 高德地图 API 密钥申请地址：https://lbs.amap.com/dev/key/app

## 浏览器兼容性

- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## 注意事项

1. **位置权限**：
   - 首次使用时，浏览器会请求位置权限，请允许以获取准确位置

2. **照片上传**：
   - 照片会在本地预览，不会自动上传到服务器
   - 实际部署时，需要在服务器端实现文件上传功能

3. **API 密钥**：
   - 高德地图 API 密钥需要替换为实际密钥才能获取详细地址信息
   - 若无 API 密钥，会显示经纬度坐标

4. **表单提交**：
   - 当前版本为前端演示版，表单数据仅在控制台输出
   - 实际部署时，需要在服务器端实现数据存储功能
